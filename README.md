## ğŸ”§ Customization Guide

### Adding Your Resume
1. Add your resume PDF to the project
2. Update the download link in `js/main.js`:
```javascript
downloadResumeBtn.addEventListener('click', (e) => {
    window.open('path/to/your/resume.pdf', '_blank');
});
```

### Updating Projects
Edit the project cards in `index.html`:
- Update titles, descriptions, and tech stacks
- Replace placeholder images with your project screenshots
- Add actual live demo and GitHub repository links

### Changing Colors
Update CSS custom properties in `css/style.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    /* ... */
}
```

### Adding More Sections
1. Add new section in `index.html`
2. Add corresponding styles in `css/style.css`
3. Update navigation menu with new link
